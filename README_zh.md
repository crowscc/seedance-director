# seedance-director

> 用即梦做视频，提示词不知道怎么写？交给你的 AI 导演。

你只需要说出创意，seedance-director 帮你完成从**故事构思 → 专业分镜 → 即梦提示词 → 平台操作**的全流程。

## 没有它 vs 有它

```
                    没有 seedance-director                    有 seedance-director
                    ─────────────────────                    ────────────────────

你的输入：          "帮我做个咖啡广告"                         "帮我做个咖啡广告"
                          │                                        │
你要做的事：        自己想故事结构                              导演问你几个选择题
                    自己查镜头术语                              自动生成专业分镜表
                    自己凑提示词                                自动输出即梦提示词
                    自己摸索平台操作                            给你逐步操作指引
                    反复试错修改                                直接粘贴就能用
                          │                                        │
花费时间：              30-60 分钟                                  3 分钟
```

## 三个核心能力

**像导演一样思考** — 不是直接丢提示词给你，而是先帮你梳理故事结构（起承转合、Hook-反转、对比型...），再拆分成专业分镜。传统影视方法论 + AI 视频，两手都硬。

**单段和长视频都能搞定** — 15 秒以内一次生成；超过 15 秒自动拆成多段，每段之间设置衔接锚点，保证画面连贯。最长支持 60 秒以上。

**从提示词到操作一条龙** — 不只给你提示词，还告诉你在即梦平台上先点哪、再传什么、参数怎么设。新手也能直接上手。

## 快速开始


## 30 秒看懂工作流

```
你：我想做一个 30 秒的咖啡广告，电影感，暖色调，有 3 张产品图。

导演：收到。叙事结构想用哪种？
      A. Hook → 揭示（蒸汽特写开场）
      B. 过程展示（咖啡豆到杯中）
      C. 生活方式（温馨场景品咖啡）

你：B

导演：
  ┌─────────────────────────────────────────────────┐
  │  分镜脚本：咖啡溯源                                │
  │  30秒（2段）| 16:9 | 电影暖调                      │
  │                                                   │
  │  001 | 0-4s  | 微距     | 缓推   | 咖啡豆纹理      │
  │  002 | 4-8s  | 近景     | 俯拍   | 倒入研磨机       │
  │  003 | 8-12s | 中景     | 固定   | 手冲，蒸汽升腾   │
  │  004 | 12-15s| 特写     | 缓推   | 咖啡液滴落       │
  │  ...                                              │
  ├─────────────────────────────────────────────────┤
  │  第 1 段提示词（直接粘贴到即梦）                     │
  │  @图片1 中的咖啡豆在晨光中缓缓滚动，微距镜头...     │
  ├─────────────────────────────────────────────────┤
  │  操作指引                                          │
  │  1. 打开即梦 → 图生视频                             │
  │  2. 上传产品图，粘贴提示词                           │
  │  3. 时长 15s，宽高比 16:9                           │
  │  4. 生成后 → 视频延长 → 粘贴第 2 段提示词           │
  └─────────────────────────────────────────────────┘
```

## 覆盖即梦全部 10 大能力

| 能力 | 一句话说明 |
|------|-----------|
| 纯文本生成 | 零素材，纯描述出视频 |
| 一致性控制 | 多镜头角色/场景不走样 |
| 运镜复刻 | 照着参考视频的镜头动 |
| 特效复刻 | 照着参考视频的特效来 |
| 剧情补全 | AI 自动补全画面过渡 |
| 视频延长 | 已有视频往后/往前续 |
| 声音控制 | 台词、音色、音效全管 |
| 一镜到底 | 多图串成不切镜长镜头 |
| 视频编辑 | 换角色、改剧情、加元素 |
| 音乐卡点 | 画面跟着节拍走 |

## 超长视频？自动拆段

| 你要的时长 | 导演怎么做 |
|-----------|-----------|
| ≤15s | 一次搞定 |
| 16-30s | 拆成 2 段，第 2 段用视频延长接上 |
| 31-45s | 拆成 3 段 |
| 46-60s | 拆成 4 段 |
| >60s | 按场景拆，各自生成后剪辑拼接 |

每段之间有**衔接锚点**：上段结尾 = 下段开头，画面不会跳。

## 内置 5 类场景模板

开箱即用，不用从零开始：

- **电商广告** — 360 度旋转、微距特写、3D 渲染
- **仙侠/动漫** — 法阵特效、变身、能量波
- **短剧/对白** — 多角色台词、情绪标注、音效设计
- **科普教学** — 可视化演示、教育旁白
- **MV/卡点** — 2.35:1 宽银幕、节拍同步、快切
