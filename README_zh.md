# seedance-director

> 用即梦做视频，提示词不知道怎么写？交给你的 AI 导演。

一个 [Agent Skill](https://agentskills.io)，在 Claude Code 里帮你走完从创意到即梦提示词的全流程。

## 安装

```bash
npx skills add https://github.com/crowscc/seedance-director
```

安装后直接聊你的视频创意。提到「即梦」「Seedance」「分镜」「运镜」「视频延长」等关键词时自动激活。

## 工作流

<p align="center">
  <img src="assets/workflow.svg" alt="seedance-director 工作流" width="700">
</p>

### Phase 1-2：理解创意 & 深度挖掘

收到你的创意后，自动扫描 5 个维度的完整度：**主题**、**时长**、**风格**、**素材**、**声音**。

缺什么问什么 — 通过 `AskUserQuestion` 弹出结构化选择题，**选项根据你的创意动态生成**。比如你说「咖啡广告」，叙事结构会优先推荐「对比型」和「过程展示」，而不是把所有选项平铺。最多问 3 轮，信息够了就直接跳到下一步。

### Phase 0：素材制备

生成分镜之前，检查你手上有没有视觉参考素材。**有就跳过，没有就帮你生成**：

- **角色三视图** — 正面 + 侧面 + 背面，锁定角色在所有镜头中的外观一致性
- **场景概念图** — 锁定环境风格、光线、色调，避免多段视频中场景风格漂移
- **关键帧** — 每段视频的首帧图，确保段与段之间画面衔接

导演会通过 `AskUserQuestion` 让你确认生成哪些素材，你也可以选择自己准备。

### Phase 3：生成分镜脚本

根据前面确定的所有信息，输出专业分镜表（景别、运镜均中英双语标注）。根据时长自动选择生成策略：

| 时长 | 策略 |
|------|------|
| **≤ 15s** | 单段生成，一次出片 |
| **16-30s** | 拆 2 段，第 2 段用**视频延长**接上（延长的最佳区间） |
| **31s+** | 每段**独立生成**，用上段末帧 + 角色三视图 + 场景图作为下段参考，避免风格漂移。各段可独立重试，最后剪映拼接 |

分镜完成后通过 `AskUserQuestion` 让你确认或调整。

### Phase 4：即梦提示词

把确认后的分镜转化为**可直接粘贴到即梦平台**的提示词。每段提示词遵循固定的**六板块结构**：

```
## 角色 + 参考图
  角色A（主角）：@图片1 — [外貌、服装描述]
  场景参考：@图片3 — [环境描述]

## 背景介绍
  [前情、环境、情绪氛围]

## 镜头描述
  镜头1（0-3s）：[景别]，[画面]，角色A："[台词]"，[运镜]
  镜头2（3-6s）：...

## 风格指令
  [统一视觉风格：质感、色调、光线、景深、帧率、宽高比]

## 禁止项
  禁止出现文字、水印、LOGO
```

每个角色独立绑定参考图，台词标注说话人，@引用使用中文并标注用途。

单段提供 2-3 个版本供选择，多段逐段输出。

### Phase 5：操作指引

即梦平台的逐步操作说明：素材上传顺序、参数设置、多段衔接流程、生成后检查要点。3 段以上还会给出剪映拼接指引。

## 30 秒看懂

```
你：我想做一个 30 秒的咖啡广告，电影感，暖色调，有 3 张产品图。

导演：[AskUserQuestion] 哪种叙事结构最合适？
      > 过程展示 — 咖啡豆到杯中的旅程，契合你的产品故事 ✓
      > 对比型 — Before/After，适合口感反差
      > 自定义 — 描述你想要的结构

你：过程展示

导演：[AskUserQuestion] 检测到缺少场景参考图，要生成吗？
      > 帮我生成场景概念图
      > 不需要，我自己准备

你：帮我生成

导演：
  场景概念图提示词 → "日出时分的咖啡庄园，金色光线，薄雾升腾..."

  分镜脚本（2 段，16:9，电影暖调）：
    001 | 0-4s  | 微距特写    | 缓推 Dolly In | 咖啡豆纹理，金色逆光
    002 | 4-8s  | 近景 CU     | 俯拍 Tilt    | 豆子倒入研磨机
    003 | 8-12s | 中景 MS     | 固定 Static  | 手冲，蒸汽升腾
    ...

  第 1 段即梦提示词（直接粘贴）：
    ## 角色 + 参考图
    角色：@图片1 — 咖啡豆，金色质感
    场景参考：@图片2 — 日出咖啡庄园

    ## 背景介绍
    从咖啡豆到杯中的旅程，金色晨光笼罩...

    ## 镜头描述
    镜头1（0-4s）：微距特写，咖啡豆纹理，缓推
    镜头2（4-8s）：近景，豆子倒入研磨机，俯拍
    ...

    ## 声音设计
    BGM：柔和钢琴 → 吉他加入 → 大提琴温暖高潮
    环境音：0-9s 键盘敲击、空调嗡鸣；9-15s 鸟鸣、引擎启动

    ## 风格指令
    电影质感，暖色调，16:9，24fps

    ## 禁止项
    禁止出现文字、水印、LOGO

  操作指引：
    上传产品图 → 设置 15s、16:9 → 生成
    → 截取末帧 → 作为第 2 段首帧参考 → 生成第 2 段
```
