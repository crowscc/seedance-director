# 场景化策略速查

> 5 大高频创作场景的核心手法和完整提示词示例。根据用户的视频类型查阅对应场景的策略。

## 场景化策略速查

### 电商/广告

**核心**：产品为主角，视觉冲击力最大化。
**策略**：360度环绕运镜 / 爆炸分解视图 / 3D渲染干净背景 / 第一人称POV沉浸 / 微距特写材质

```
商业广告质感，高端产品摄影

0-3秒：@图片1 中的香水瓶从虚焦中逐渐清晰，缓推 Dolly In，
金色液体在瓶内微微晃动，体积光从背后打入
4-7秒：镜头 360 度环绕香水瓶旋转，展示瓶身雕刻细节，
光影在瓶面流转，微粒子飘散
8-10秒：香水喷射出金色雾气，雾气在空中形成品牌标志性图案，
镜头缓拉，黑色背景，高级感

@图片1 作为产品实物形象
禁止出现文字、水印、LOGO
```

### AI漫剧/仙侠

**核心**：角色一致性 + 特效场面 + 连贯叙事。
**策略**：@图片锁定角色形象 / 时间戳分镜控制节奏 / 特效描述要具体化 / 多段延长注意服装发型一致

```
3D国漫CG风格，电影级光影

0-3秒：@图片1 中的白衣剑客站在悬崖边，长发飘动，
俯瞰云海翻涌的深渊，远景 Wide Shot，侧逆光勾勒轮廓
4-8秒：剑客拔剑，剑身发出青色光芒，剑气如游龙盘旋而出，
镜头从中景快速推至近景，浅景深虚化背景
9-12秒：纵身跃下悬崖，衣袂飘飞，镜头高速跟拍下坠，
云雾从身边掠过，慢动作，体积光穿透云层

@图片1 作为主角形象参考（白衣、长发、佩剑）
禁止出现文字、水印
```

### 短剧/对白

**核心**：表演自然 + 台词清晰 + 情绪到位。
**策略**：画面与台词分行写 / 标注角色情绪 / 音效单独描述 / 对话用中景、反应用近景

```
电影写实风格，自然光

0-4秒：中景 Medium Shot，@图片1 中的女孩坐在咖啡桌对面，
表情从犹豫变为坚定，缓推至近景
台词（温柔但坚定）："我想了很久，我决定去北京。"
5-7秒：切换反打镜头，@图片2 中的男孩愣住，
手中的咖啡杯停在半空，近景 Close-Up
音效：咖啡馆环境音，轻微的杯碟声
8-10秒：两人对视，沉默，窗外阳光缓缓移动
音乐：钢琴轻音乐渐入

@图片1 作为女主角形象参考
@图片2 作为男主角形象参考
禁止出现文字、水印
```

### 科普教学

**核心**：信息清晰 + 视觉化复杂概念 + 可信度。
**策略**：医学CGI可视化 / 半透明可视化 / 旁白与画面时间戳同步 / 微距横截面分层展示

```
科学纪录片质感，专业医学CGI

0-4秒：人体心脏的3D模型缓慢旋转，半透明显示内部结构，
镜头缓推 Dolly In，黑色背景，冷白色专业照明
旁白："人类的心脏每天跳动约十万次"
5-10秒：镜头穿透心壁进入心室内部，红色血流涌动，
瓣膜开合的慢动作特写，体积光照亮血管壁纹理
旁白："每次跳动推动约70毫升血液"
11-15秒：镜头拉出，心脏恢复正常跳动节奏，
血管网络从心脏向全身延展的可视化动画
旁白："一天泵出超过7000升血液，足以填满一辆油罐车"

禁止出现文字、水印、LOGO
```

### MV/音乐卡点

**核心**：视觉节奏 = 音乐节奏，画面冲击力。
**策略**：宽银幕构图 / 分镜切换对齐节拍 / 强调色彩对比和动态转场 / @音频引用BGM

```
MV风格，高饱和色彩，宽银幕构图，电影质感

参考 @音频1 的节奏，画面切换跟随鼓点：
0-2秒（第一拍）：@图片1 从暗场中亮起，霓虹紫光，快速推镜
2-4秒（第二拍）：硬切到 @图片2，荷兰角构图，闪烁频闪效果
4-6秒（第三拍）：@图片3 慢动作旋转，金色粒子飞溅
6-8秒（第四拍）：快速交叉剪辑 @图片1 和 @图片4，节奏加倍
8-10秒（收尾）：所有画面叠化融合，光芒爆发后渐暗

@图片1 @图片2 @图片3 @图片4 作为四个分镜的画面素材
@音频1 作为BGM节奏参考
禁止出现文字、水印
```
